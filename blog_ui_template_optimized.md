# 🚀 Blog UI Template – Bản Tối Ưu Để Bán

## 🎯 Mục tiêu
Tạo một template blog:
- Nhìn chuyên nghiệp
- Có structure rõ ràng
- Dễ bán cho khách (doanh nghiệp / dev)

---

## 🧩 Layout Tối Ưu (Khuyến nghị)

```
[ HERO ]

[ FEATURED POST ]

[ CATEGORIES ]

[ LATEST POSTS ] + SIDEBAR

[ SERIES / LEARNING PATH ]

[ NEWSLETTER ]

[ CTA ]

[ FOOTER ]
```

---

## 🔥 1. Hero Section (BẮT BUỘC)

### Nội dung:
- Title lớn
- Subtitle
- 2 CTA buttons

### Ví dụ:

```
🚀 Chia sẻ kiến thức Backend & DevOps
Xây dựng hệ thống scalable, production-ready

[ Khám phá bài viết ]   [ Liên hệ ]
```

### Design:
- Background: gradient (dark → blue)
- Text: 32–40px
- Padding lớn

---

## 🥇 2. Featured Post

### Mục tiêu:
Highlight 1 bài viết quan trọng

### UI:
- Ảnh lớn
- Title to
- Description ngắn

---

## 🧠 3. Categories Section

### Nội dung:
- Danh sách chủ đề

### Ví dụ:
```
[ AI ] [ Backend ] [ DevOps ] [ Cloud ]
```

### Tips:
- Click để filter bài viết
- Dùng dạng pill (rounded)

---

## 📰 4. Latest Posts (Section chính)

### Bao gồm:
- Grid bài viết (3 cột)
- Sidebar

### Sidebar nên có:
- Bài viết mới
- Tags
- Newsletter nhỏ

---

## 📚 5. Series / Learning Path

### Rất quan trọng với blog tech

### Ví dụ:
- Kafka từ A-Z
- Spring Boot Production
- Docker cho người mới

### UI:
- Card hoặc list

---

## 📧 6. Newsletter (Full width)

### Ví dụ:
```
📬 Nhận bài viết mới mỗi tuần
[ Email ] [ Đăng ký ]
```

---

## 🎯 7. CTA (Convert khách)

### Ví dụ:
```
Bạn cần xây dựng hệ thống backend?
[ Liên hệ ngay ]
```

---

## 🎨 Design System

### Colors:
- Primary: #2563eb
- Background: #0f172a (dark)
- Card: #1e293b

### Typography:
- Font: Inter
- Title: bold
- Body: 16px

### Card:
- Border radius: 12–16px
- Shadow nhẹ
- Hover: scale + shadow

---

## ⚡ Hiệu ứng cần có

```
transition: all 0.3s ease;
```

- Hover card
- Hover button
- Image zoom
